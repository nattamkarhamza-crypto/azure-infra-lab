# Azure Infra Lab

Personal Azure learning lab built to develop and document hands-on cloud engineering skills.

## What This Lab Covers
- Azure VM management and automation
- Entra ID user lifecycle scripting
- VNet, NSG, and VPN configuration
- ARM template deployments
- Azure Monitor and cost reporting
- Incident runbooks and SOPs

## Repository Structure
azure-infra-lab/
├── scripts/        # PowerShell automation scripts
├── arm-templates/  # ARM deployment templates
├── docs/           # Runbooks, SOPs, architecture diagrams
└── README.md
